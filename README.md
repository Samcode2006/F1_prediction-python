# 🏎️ F1 Race Winner Prediction System

A beginner-friendly Python mini project that predicts a Formula 1 race winner using rule-based scoring logic. Now features live data fetching via `fastf1` and an interactive web dashboard via `streamlit`.

---

## 📁 Folder Structure

```
F1_Predictor/
│
├── main.py          ← CLI Entry point. Runs the program with a menu in terminal.
├── app.py           ← Streamlit UI Entry point. Runs the interactive web dashboard.
├── fetch_data.py    ← Script to fetch real F1 data for the current season using FastF1.
├── predictor.py     ← Core logic: load, score, rank drivers.
├── drivers.csv      ← Input data: driver grid positions & past results (generated by fetch_data.py).
└── README.md        ← This file.
```

---

## 🚀 How to Install and Run

### 1. Install Dependencies
You need a few external libraries for the data fetching and web interface:
```bash
pip install pandas fastf1 streamlit
```

### 2. Fetch Latest Data
Before running the predictor, generate the `drivers.csv` file using real data:
```bash
python fetch_data.py
```

### 3. Run the CLI Version
```bash
python main.py
```

### 4. Run the UI Version (Web Dashboard)
```bash
streamlit run app.py
```

## 📊 How the Scoring Works

Each driver gets a **prediction score** based on three factors:

| Factor             | Logic                                          | Max Points |
|--------------------|------------------------------------------------|------------|
| Qualifying Position | P1 → 10 pts, P2 → 9 pts, …                   | 10         |
| Previous Race Finish | P1 → 10 pts, P2 → 9 pts, …                  | 10         |
| Podium Form Bonus  | +5 pts if finished P1–P3 in last race          | 5          |
| Team Strength      | Based on constructor rating (1–10 scale)       | 10         |

**Highest total score = Predicted Winner**

## 📝 CSV Format

```csv
Driver,Team,Qualifying,PreviousFinish
Verstappen,Red Bull,1,2
Hamilton,Mercedes,4,3
```

- `Qualifying` — starting grid position (1 = pole)
- `PreviousFinish` — finishing position in the last race (1 = win)
